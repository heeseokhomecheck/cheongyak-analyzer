## 갱신 방법 (새 대책 발표 시)
1. 감시기(매일)가 **금융위·국토부 공식 보도자료** 새 글 감지 → GitHub 이슈 알림
2. 보도자료 **원문 확인**
3. 이 표에서 해당 줄 찾기 → `index.html`의 **`POLICY_LOAN`/`RULES_ASSET`/`acqTax` 등 해당 상수**만 수정 + `POLICY_ASOF` 날짜 갱신
4. 커밋·푸시 → 자동 배포

> ⚠️ 표시(가정) 항목은 단정하지 않고 "가정·확인"으로 안내 중. 확정 출처를 찾으면 ✅로 승격.
